# Eff_Eff_Spcial!!
えふえふスペシャル！！ポン出しツールです
とりあえずほぼAI製なので後でブラッシュアップかける予定

## How to use
- `Folder`: 音声フォルダを選択します。上から順に最大80個、A-Eの5チャンネル x 16パッドへ割り当る
- `5 6 7 8 9`: チャンネルA-Eを切り替える
- `1 2 3 4 / q w e r / a s d f / z x c v`: 表示中チャンネルのパッドを鳴らす
- `0`: 全音を即停止
- 各パッドの `STOP`: そのパッドを即停止
- 各パッドの `TAPE`: そのパッドをテープストップ
- `Shift + パッドキー`: そのパッドを即停止
- `Alt + パッドキー`: そのパッドをテープストップ

取り込んだ音声は、対応ブラウザではIndexedDBに保存され、次回起動時に復元される（はずだよ♡）
